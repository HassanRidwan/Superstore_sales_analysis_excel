# Superstore_sales_analysis_excel
Excel dashboard analyzing sales performance and discount impact on profitability
# Project Overview
This project analyzes sales performance using the Superstore dataset. An interactive Excel dashboard was developed to track key business metrics and uncover insights across segments, regions, and product categories.
# Tools Used
Microsoft Excel

PivotTables

Data Visualization (Charts & Dashboard Design)
# Key KPIs
Total Sales

Total Profit

Total Quantity Sold

Total Orders

Profit Margin %

Average Discount %

# Key Insights
Consumer segment contributes the highest share of total sales

Technology category generates the highest profit

Sales show strong seasonal trends across months

High discount levels lead to losses, while low discounts drive most of the profit

Aggressive discounting negatively impacts overall profitability
# Conclusion
The analysis highlights the importance of optimizing discount strategies to improve profitability while maintaining sales performance.
